Emotion Analysis on Twitter Data

Data Source for Tweets: Mined tweets using GetOldTweets3 python library.

Data Source for Covid-19 cases: COVID19-India API | api.covid19india.org.

The main purpose of this web portal is to provide results of analysis of public emotions during Covid-19 based on twitter data. Disclaimer: The emotions displayed here are solely dependent on the classification technique used, as a part of academic exercise. We are not responsible for the accuracy of emotions. This portal is continuously updated on a daily basis, with an average of 10K tweets per day.
